Gerenciador de Tarefas

Defina; Organize; Realize.

Está aplicação é um gerenciador de tarefas. O usuário cadastra um projeto e especifica as etapas que levarão à sua conclusão.

Quando uma tarefa é concluída, o usuário pode sinalizá-la como realizada e partir para a próxima.

O objetivo da aplicação é facilitar o desenvolvimento de atividades, de forma que o cumprimento das etapas levem à conclusão do projeto de forma estruturada e objetiva.

Este projeto foi realizado para ser submetido à avaliação, como projeto final.

O código fonte está disponível neste repositório, junto dos requisitos definidos e orientações para execução.
